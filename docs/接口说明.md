# 接口说明

## 基础说明

- 后端默认地址：`http://localhost:8080`
- 接口统一以前缀 `/api` 暴露。
- 前端通过 Axios 请求封装调用后端，登录后携带 token。
- 本文列出客户演示和二次开发最常用接口，完整接口以 Controller 代码为准。

## 认证与用户

| 方法 | 地址 | 说明 |
| --- | --- | --- |
| `POST` | `/api/auths/login` | 登录 |
| `DELETE` | `/api/auths/logout` | 登出 |
| `POST` | `/api/auths/register` | 注册 |
| `GET` | `/api/auths/captcha` | 获取验证码 |
| `GET` | `/api/user/info` | 当前用户信息 |
| `GET` | `/api/user/paging` | 用户分页 |
| `POST` | `/api/user` | 新增用户 |
| `PUT` | `/api/user` | 更新用户 |
