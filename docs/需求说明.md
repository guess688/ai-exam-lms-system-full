# 需求说明

## 项目名称

AI 智能考试与网课学情分析系统

## 项目定位

本项目是在现有 Spring Boot + Vue 在线考试系统基础上的二次开发项目，面向教学考试、网课学习、学情分析和教师备课场景。系统不推翻原考试系统架构，优先复用登录、角色、班级、题库、试卷、考试、判分、成绩、错题本等既有能力，在其上新增课程体系、网课进度、学习任务和 AI 服务。

## 用户角色

| 角色 | 主要能力 |
| --- | --- |
| 管理员 | 管理用户、班级、基础数据和系统配置 |
| 教师 | 管理课程、章节、知识点、题库、试卷、考试、网课、学习任务、成绩分析和 AI 功能 |
| 学生 | 参加考试、查看成绩、学习网课、完成任务、查看个人学情报告 |

## 功能范围

### 已复用能力

- 登录、注册、用户信息和权限控制
- 班级管理
- 题库、试题、选项管理
- 试卷和考试发布
- 在线答题、自动判分、成绩记录
- 错题本和刷题记录
- 成绩查询和基础统计

### 新增能力

- 课程、章节、知识点三层教学内容模型
- 题目绑定课程、章节、知识点和难度
- 按课程、章节、知识点、题型、难度自动组卷
- 考后章节掌握、知识点掌握、错题和难度表现分析
- 教师端和学生端图表展示
- 轻量网课系统：视频、资料、观看进度、完成率
- 学习任务系统：考试、练习、网课、错题订正、复习
- AI 服务基础设施：mock 模式和真实模型配置
- AI 学生个人学情报告
- AI 班级学情报告
- AI 自动出题和教师审核入库
- AI 推荐练习任务
- 学习行为评价

## 明确不优先实现

- 家校沟通反馈
- 简答题 AI 评分的深度优化
- 防作弊增强方案
- 直播课
- 小程序端

## 核心业务规则

- 教师只能管理自己创建或负责班级内的课程、试卷、任务和分析数据。
- 学生只能查看自己班级可见的课程、考试、任务和报告。
- 难度统一为 `EASY`、`MEDIUM`、`HARD`。
- 不定项选择题暂按完全一致判分，不做半分。
- AI 生成题目必须先进入待审核列表，教师确认后才进入正式题库。
- AI API Key 不允许写死在代码中，必须通过配置文件占位或环境变量注入。

## 验收重点

- 新开发者按照 README 能启动前后端。
- 演示账号可登录。
- 演示数据能覆盖考试、成绩、章节掌握、网课进度、任务和 AI 功能。
- 原有考试流程不被破坏。
- mock AI 模式下可完整演示所有 AI 功能。
