# 数据库设计

## 初始化顺序

建议按以下顺序初始化：

```bash
mysql -uroot -p db_exam < backend/sql/db_exam.sql
mysql -uroot -p db_exam < backend/sql/20260628_course_content.sql
mysql -uroot -p db_exam < backend/sql/20260629_question_course_difficulty.sql
mysql -uroot -p db_exam < backend/sql/20260629_online_course.sql
mysql -uroot -p db_exam < backend/sql/20260629_learning_task.sql
mysql -uroot -p db_exam < backend/sql/20260630_ai_infrastructure.sql
mysql -uroot -p db_exam < backend/sql/20260630_ai_report.sql
mysql -uroot -p db_exam < backend/sql/20260630_ai_generated_question.sql
mysql -uroot -p db_exam < backend/sql/20260630_demo_data.sql
```

## 原有核心表

| 表名 | 用途 |
| --- | --- |
| `t_user` | 用户账号、角色、班级归属 |
| `t_role` | 角色定义 |
| `t_grade` | 班级 |
| `t_user_grade` | 教师与班级关系 |
| `t_category` | 题库分类 |
| `t_repo` | 题库 |
| `t_question` | 试题 |
| `t_option` | 试题选项 |
| `t_exam` | 考试/试卷基础信息 |
| `t_exam_question` | 考试题目关联 |
| `t_exam_grade` | 考试和班级关联 |
| `t_exam_qu_answer` | 学生答题记录 |
| `t_user_exams_score` | 学生成绩 |
| `t_user_book` | 错题本 |

## 教学内容模型

| 表名 | 说明 |
| --- | --- |
| `t_course` | 课程，包含名称、简介、封面、教师、班级、状态 |
| `t_course_chapter` | 课程章节，按课程和排序组织 |
| `t_knowledge_point` | 知识点，绑定课程和章节 |

`t_question` 通过迁移脚本新增：

| 字段 | 说明 |
| --- | --- |
| `course_id` | 课程 ID |
| `chapter_id` | 章节 ID |
| `knowledge_point_id` | 知识点 ID |
| `difficulty` | `EASY`、`MEDIUM`、`HARD` |

## 网课模型

| 表名 | 说明 |
| --- | --- |
| `t_course_video` | 课程视频，绑定课程和章节 |
| `t_course_material` | 课程资料，绑定课程和章节 |
| `t_student_video_progress` | 学生视频进度，记录观看秒数、进度率、完成状态 |

完成规则：`progress_rate >= 90` 视为完成。

## 学习任务模型

| 表名 | 说明 |
| --- | --- |
| `t_learning_task` | 教师发布的学习任务 |
| `t_learning_task_record` | 学生任务完成记录 |

任务类型：

| 值 | 含义 |
| --- | --- |
| `EXAM` | 考试任务 |
| `PRACTICE` | 练习任务 |
| `VIDEO` | 网课任务 |
| `WRONG_QUESTION` | 错题订正任务 |
| `REVIEW` | 复习任务 |

## AI 模型

| 表名 | 说明 |
| --- | --- |
| `t_ai_call_log` | AI 调用日志，记录功能类型、模型、mock 状态、耗时、成功状态和脱敏摘要 |
| `t_ai_question_review` | AI 出题审核队列，兼容题库入库结构 |
| `ai_generated_question` | AI 生成题目审核表 |
| `ai_report` | AI 报告，保存输入 JSON、输出文本和结构化结果 |

## 演示数据

`backend/sql/20260630_demo_data.sql` 使用 `9000` 段 ID，包含：

- 演示账号和演示班级
- 演示课程、章节、知识点
- 演示题库、题目和选项
- 演示考试、答题记录和成绩
- 演示错题记录
