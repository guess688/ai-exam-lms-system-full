# 演示流程

## 演示准备

1. 按 README 完成数据库、后端和前端启动。
2. 执行 `backend/sql/20260630_demo_data.sql`。
3. 使用 mock AI 模式，避免演示时依赖外部模型。
4. 推荐浏览器地址：`http://localhost:9528`。

## 演示账号

| 角色 | 用户名 | 密码 |
| --- | --- | --- |
| 管理员 | `admin` | `123456` |
| 教师 | `teacher` | `123456` |

| 学生演示号 | `demo_student` | `123456` |

## 标准演示路径

1. 管理员登录，查看用户和班级。
   - 登录 `admin / 123456`。
   - 进入用户管理和班级管理。
   - 展示管理员可维护基础账号、角色和班级数据。

2. 教师创建课程、章节、知识点。
   - 登录 `teacher / 123456` 或 `demo_teacher / 123456`。
   - 进入课程管理，查看或新建“初中数学 AI 演示课”。
   - 进入章节管理，展示“一元一次方程”“几何初步”。
   - 进入知识点管理，展示“等式性质”“一元一次方程求解”“平行线性质”。

3. 教师新增题目，设置难度、章节、知识点。
   - 进入题库管理或题目管理。
   - 新增单选、多选、不定项选择、判断题。
   - 展示课程、章节、知识点联动选择。
   - 展示难度下拉框：简单、中等、困难。

4. 教师自动组卷并发布考试。
   - 进入考试管理或创建考试页面。
   - 选择自动组卷。
   - 设置课程、章节、知识点、题型数量和难度。
   - 生成并发布“AI 演示章节测验”。

5. 学生在线考试并提交。
   - 登录 `student / 123456` 或 `demo_student / 123456`。
   - 进入考试中心。
   - 打开“AI 演示章节测验”，完成答题并提交。
   - 展示系统自动判分。

6. 教师查看成绩和图表。
   - 教师进入成绩统计或考试分析页面。
   - 查看班级成绩分布、分数段统计、学生成绩列表。
   - 展示 ECharts 图表加载和空状态处理。

7. 教师查看章节/知识点掌握分析。
   - 在考试分析页查看章节正确率、知识点正确率。
   - 查看薄弱章节排名、薄弱知识点排名、高频错题排名。
   - 查看不同难度题目的正确率。

8. 教师发布网课任务。
   - 进入网课视频管理，查看“等式性质精讲”“平行线性质突破”。
   - 进入学习任务管理。
   - 发布或查看“观看平行线性质突破视频”任务。

9. 学生观看网课并产生进度。
   - 学生进入我的课程。
   - 打开课程详情，播放章节视频。
   - 等待进度上报，查看视频学习进度。
   - 完成率达到 90% 后视为完成。

10. 教师生成 AI 班级学情报告。
    - 教师进入考试分析页面。
    - 点击生成 AI 班级学情报告。
    - 展示班级整体表现、薄弱章节、薄弱知识点、高频错题、教学建议。
    - 强调报告不包含家校沟通反馈。

11. 学生生成 AI 个人学情报告。
    - 学生进入成绩详情页。
    - 点击生成 AI 学情报告。
    - 展示整体表现、优势章节、薄弱知识点、错题原因、网课学习建议和推荐练习方向。

12. 教师使用 AI 自动出题，审核后入库。
    - 教师进入 AI 待审核题目页面。
    - 选择课程、章节、知识点、题型、难度、数量。
    - 生成 AI 题目。
    - 编辑题干、选项、答案和解析。
    - 点击审核入库，确认题目进入正式题库。

13. 教师生成 AI 推荐练习并发布任务。
    - 教师在考试分析页面点击生成推荐练习。
    - 查看推荐章节、知识点、题目列表、推荐理由和建议截止时间。
    - 编辑推荐题目。
    - 一键发布为学习任务。
    - 学生在任务列表中看到推荐练习任务。
